 Student Performance Analysis Dashboard – Power BI Project

 Project Overview

This project is an interactive Power BI dashboard developed using the Portuguese Student Performance dataset from the [UC Irvine Machine Learning Repository](https://archive.ics.uci.edu/?utm_source=chatgpt.com).

The dashboard focuses on analyzing student academic performance across:

* Grade 1 (G1)
* Grade 2 (G2)
* Grade 3 (G3)

The main objective of this project is to transform raw educational data into meaningful insights using data visualization and business intelligence techniques.

---

 Dashboard Features

* Analysis of G1, G2, and G3 student grades
* Absence vs academic performance analysis
* Internet access impact on student results
* Study support and paid classes analysis
* Family relationship and family size analysis
* Parent occupation insights
* School selection reason analysis
* KPI cards showing:

  * Average Grade
  * Minimum Grade
  * Maximum Grade
* Interactive visualizations:

  * Bar Charts
  * Donut Charts
  * Scatter Plots
  * Matrix Tables

 Tools & Technologies Used

* Power BI
* Power Query
* Data Cleaning
* Data Transformation
* Data Visualization
* Dashboard Design



Key Insights

* Students with fewer absences generally performed better.
* Internet access showed an influence on academic performance.
* Study support and paid classes impacted grade improvement.
* Family relationships and previous failures affected student outcomes.
